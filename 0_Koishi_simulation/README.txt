こちらのディレクトリは、上記のrefにある福井大の小石先生の演習の回答となるコードの例を示しています。
あくまでも一例であり、より良いコーディング方法があることには注意してください。

- 各プログラムは以下のようにコンパイルすることを推奨します。
  ` $ gcc -Wall ***.c `
- 結果として`a.out`という名前の実行ファイルが作成されますので、それをターミナル上で以下のように実行してください。
  `$ ./a.out`
- 出力ファイルは全て`output`ディレクトリに格納されるように設定しています。
